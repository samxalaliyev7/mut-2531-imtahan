# Təhlükəsizlik qeydi

Bu versiya GitHub Pages üçün statik MVP-dir.

Statik saytda backend və database olmadığı üçün bütün nəticələr tələbənin öz cihazında `localStorage`-da saxlanır. Bu, pilot yoxlama və təqdimat üçün uyğundur, amma real imtahan nəzarəti üçün kifayət deyil.

Real imtahan üçün tövsiyə olunan əlavə funksiyalar:

- Server-side login
- Database-də nəticələrin saxlanması
- Admin paneldə real vaxtda bütün nəticələrin görünməsi
- Hər tələbə üçün yalnız bir cəhd limiti
- İmtahan aktiv/deaktiv rejimi
- Sessiya və token təhlükəsizliyi
